# MERN Stack App

## 📁 Structure

- `/frontend` – React app
- `/backend` – Express/MongoDB app

## 🚀 How to Run

```bash
# frontend
cd frontend
npm install
npm run dev

# backend
cd backend
npm install
npm run dev
```

## Also make a .env file inside app folder in backend in below manner

```bash
MONGODB_URI= Your Mongodb URI
JWTKEY = Your JWT key
PORT - Your Port number
```
